# 准备工作

# Node.js

注意

需要 Node.js 版本 >= 8.10.0

验证 node 当前版本

$ node -v
# v8.10.x

# Yarn

建议

使用 yarn 代替 npm 管理 node 依赖

全局安装 yarn

$ npm i -g yarn

验证 yarn 是否安装成功

$ yarn -v
# 1.19.1

# 淘宝镜像

建议

安装 node 依赖之前，设置淘宝镜像

查询 npm 镜像仓库

$ npm get registry
# https://registry.npmjs.org/

查询 yarn 镜像仓库

$ yarn config get registry
# https://registry.npmjs.org/

设置 npm 淘宝镜像

$ npm set registry http://registry.npm.taobao.org/

设置 yarn 淘宝镜像

$ yarn config set registry http://registry.npm.taobao.org/
